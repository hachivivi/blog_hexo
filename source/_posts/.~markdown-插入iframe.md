# markdown 插入iframe

markdown是兼容html的，所以使用和html一样

```html
<iframe src="https://baidu.com" width="700px" height="500px" frameborder="0" scrolling="no"> </iframe>
```

效果展示：

<iframe src="https://baidu.com" width="700px" height="500px" frameborder="0" scrolling="no"> </iframe>

注意事项：

1. & -> &amp


2. <></>中间要留有空格
